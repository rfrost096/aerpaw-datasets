# LaTeX Presentation Generation

This is the procedure for generating a presentation on the dataset analysis.
OS system used: Ubuntu 24.04.4 LTS
IDE: VSCode

## Install Procedure

- sudo apt update
- sudo apt install texlive-latex-recommended texlive-latex-extra texlive-fonts-recommended latexmk
- Optional: add LaTeX Workshop by James Yu to VSCode

## Compile Procedure

- Using LaTeX Workshop by James Yu
